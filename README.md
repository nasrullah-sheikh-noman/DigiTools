# 🚀 Project Name : DigiTools

## 📖 Description
DigiTools is a modern web application that provides a collection of useful digital tools in one place. Users can browse different products, add them to a cart, manage selected items, and interact with a smooth and responsive interface. This project demonstrates practical frontend development skills using React and modern UI frameworks.

---

## 🛠️ Technologies Used
- React.js
- JavaScript(ES6+)
- Tailwind CSS
- Vite
- DaisyUI
- HTML5

---

## ✨ Features
- 🔹 Adding card in cart options.
- 🔹 Button toggling Product and carts button and show cards list & carts list.
- 🔹 Delete card in carts list and delete all card in carts list using clear all carts button.
- 🔹 React toastify using clicking all interactivity buttons.
- 🔹 Fully Responsive for all device.

## 📂 Installation & Setup
Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/nasrullah-sheikh-noman/DigiTools.git

# Go to project folder
cd your-repo-name

# Install dependencies
npm install

# Run the project
npm run dev
```
